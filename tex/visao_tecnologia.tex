\section{Visão Tecnologia}
São premissas do projeto que os servidores de aplicação terão Ubuntu Linux 16.04 instalados, e que os aparelhos para instalação da aplicação de validação têm Android 4.4 ou superior.

As aplicações de servidor serão escritas em Python 3.6, utilizando-se o framework Django como middleware para as funções de comunicação via HTTP, autenticação e permissão de usuários. As bibliotecas Scipy e Numpy serão utilizadas para a realização de computações numéricas contidas nos modelos estatísticos.

O SGBD utilizado será o PostgreSQL, devido a seu baixo custo e performance aceitável.
